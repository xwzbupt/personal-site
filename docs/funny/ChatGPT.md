# **青龙面板**

::: tip

科学上网是第一步，需要提前准备。将节点设置为ChatGPT支持的国家，比如美国。

:::

## **一、短信接收平台**

进入https://sms-activate.org，注册一个账户，QQ邮箱即可，密码包含字母大小写和数字。

## 充值

![image-20230120213631276](C:/Users/Weiser/AppData/Roaming/Typora/typora-user-images/image-20230120213631276.png)

选择支付宝充值，充0.2美元应该就够了，我这里充了0.25美元。

![image-20230120213957349](C:/Users/Weiser/AppData/Roaming/Typora/typora-user-images/image-20230120213957349.png)

充值完毕以后，在网页的右侧搜索openai，选择一个国家的虚拟号收验证码。这里我选择的是马来西亚。记住，点击购物车就代表向虚拟手机号发送验证码，轻易不要点，确认再点。

![image-20230120215615781](C:/Users/Weiser/AppData/Roaming/Typora/typora-user-images/image-20230120215615781.png)

如上图所示，这里申请的虚拟号码，状态是等待短信。

这时进入openai的官网的登录界面，选择马来西亚，将60后面的号码填进去，点击发送验证码。这时会收到验证码，再将验证码填入openai的网站中。

![image-20230120215733355](C:/Users/Weiser/AppData/Roaming/Typora/typora-user-images/image-20230120215733355.png)
